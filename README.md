# Blog Aggregator

A simple RSS feed aggregator for reading my favorite blogs in one place.

## Features
- Aggregates posts from 5 blogs
- Mark posts as read/unread
- Read status persists across sessions using localStorage
- Clean, responsive design
- Mobile-friendly

## Live Site
https://agregadorblogs.netlify.app/

## Blogs Included
- Rodobo
- Shreyas Doshi
- Simon Sarris
- Henrik Karlsson
- Bonillaware

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Deployed on Netlify with automatic deployment from GitHub

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. That's it! No build process needed.

## Deployment
This site automatically deploys to Netlify when changes are pushed to the `main` branch.
